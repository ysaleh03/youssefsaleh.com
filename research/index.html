research experience

Undergraduate Research Assistant

May 2026 – Aug 2026

UBC Software Practices Lab

  • Supervisor: Dr. Alexander J. Summers, funded by UBC EnSURE
  • Developing a new prototype implementation of Mendel, a technique for verifying clients of interior-mutable Rust modules that uses implicit capabilities describing the access permissions granted by the design of module APIs.
  • Characterizing the interaction between Mendel’s implicit capability model and the place capability information exposed by Place Capability Graphs, a state-of-the-art model of the Rust compiler’s type checking results.

Undergraduate Research Assistant + Honours Thesis

Apr 2025 – Aug 2025

UBC Software Practices Lab

  • Supervisor: Dr. Alexander J. Summers, partially funded by UBC SURE
  • Designed and implemented a novel encoder for Prusti, a Viper-based Rust verifier, using only pure assertions to eliminate modelling heap memory explicitly for quicker, more efficient verification.
  • Rewrote function call modelling for Place Capability Graphs to produce a more precise representation of Rust references and reborrowing semantics.
